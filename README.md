# 🏆 Quiz Master - Advanced JavaFX Application

## 📝 Description
**Quiz Master** is a sophisticated Full-Stack quiz application built with **Java 25**. It features a modern, user-friendly interface and a **Smart Difficulty Algorithm** that adapts the challenge level based on player performance.

This project demonstrates advanced skills in **Object-Oriented Programming (OOP)**, database management, and professional UI design.

## 🚀 Key Features
* **🎨 Modern Dashboard:** Intuitive navigation using category cards (Arts, Science, Manga, etc.).
* **🧠 Adaptive Difficulty Engine:** A custom algorithm that tracks success rates to adjust question complexity.
* **📊 User Analytics:** Detailed performance tracking with historical data and score visualization.
* **🔐 Admin Back-office:** A secure area to manage the entire database (Questions, Themes, Users) with full CRUD capabilities.

## 🛠️ Technical Stack
* **Language:** Java (JDK 25)
* **Framework:** JavaFX (Scene Builder + CSS styling)
* **Database:** MySQL (JDBC Connection)
* **Build Tool:** Maven (Wrapper included for easy setup)

## ⚙️ Installation & Setup

### 1. Prerequisite
* **Java JDK 25** installed.
* **MySQL Server** (via WampServer, XAMPP or Docker).

### 2. Database Configuration
Import the `quiz_ece.sql` file into your local MySQL environment to set up the tables and sample questions.
